# GreenLens — 景观设计智能辅助平台 MVP v0.1

基于 Web 的景观设计辅助工具，支持分色卫星图导入、语义点阵建模、手动标签编辑与绿地空间分析。

## 本地运行

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev

# 3. 浏览器访问
# http://localhost:5173
```

**环境要求**: Node.js 18+，Chrome / Edge（推荐）

---

## 功能说明

### 1. 导入分色图
- 拖放或点击上传 PNG / JPG 分色卫星图
- 自动执行 K-means（k=8）聚类提取主色调
- 弹出「颜色→标签映射」面板，用户确认配对后建立底图

### 2. 标签编辑
| 工具 | 操作 |
|------|------|
| 🖌 笔刷 | 按住鼠标拖拽，实时涂抹修改标签 |
| ⬡ 多边形 | 点击添加顶点，双击或点击起点完成选区 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |

### 3. 面板说明
- **图例**：标签颜色图例 + 操作说明
- **统计**：各类用地面积（hm²）与占比
- **分析**：绿地服务圈覆盖率 + 优化建议

### 4. 绿地分析
点击工具栏「◎ 绿地分析」按钮：
- BFS 计算每格元到最近绿地的距离
- 生成 300 / 500 / 1000 m 服务圈覆盖率
- 叠加热力图（绿→黄→红）
- 基于规则生成优化建议文字

### 5. 数据持久化
点击「💾 保存」将点阵数据存入浏览器 **IndexedDB**，刷新后可恢复。

## 数据模型

```
RasterGrid {
  width: number        // 格元列数
  height: number       // 格元行数  
  cellSize: number     // 格元尺寸（像素/米）
  data: Uint8Array     // 每格元一个标签 ID (0-7)
}
```

## 标签体系
| ID | 类别 | 颜色 |
|----|------|------|
| 0 | 未分类 | 灰色 |
| 1 | 建筑 | 橙色 |
| 2 | 道路/硬质 | 浅灰 |
| 3 | 绿地/植被 | 绿色 |
| 4 | 水体 | 蓝色 |
| 5 | 农田 | 黄绿 |
| 6 | 裸地 | 棕色 |
| 7 | 其他用地 | 紫色 |

## 技术栈
- **框架**: React 18 + Vite
- **状态**: Zustand
- **渲染**: HTML5 Canvas（语义点阵）+ OffscreenCanvas
- **聚类**: 纯 JS K-means++（无外部依赖）
- **空间分析**: BFS 欧式距离（无外部依赖）
- **持久化**: IndexedDB
